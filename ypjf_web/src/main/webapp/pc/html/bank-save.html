拥抱监管 主动合规

依据2016年8月24日由银监会、工信部、公安部、网信办联合公布的《网络借贷信息中介机构业务活动管理暂行办法》要求，需要互联网金融平台与银行业金融机构对接，对平台资金进行银行存管，具体做法是互联网金融平台在银行业金融机构开设独立账户，用于存放用户的指令、合同内容，对用户的交易与资金流水进行划拨和簿记，以保障用户的资金安全。之后，银监会又在《网络借贷资金存管业务指引》中指明：网络借贷资金存管业务，是指商业银行作为存管人接受委托人的委托，按照法律法规规定和合同约定，履行网络借贷资金存管专用账户的开立与销户、资金保管、资金清算、账务核对、提供信息报告等职责的业务。

  • 独立账户

    内蒙古陕坝银行为投资人开设独立的存管账户，与平台资金完全隔离，资金流向全程由银行监管，杜绝资金池。

  • 用户授权

    银行根据投资人指令对账户资金进行划拨，交易全部在银行存管系统内执行，平台无法调配资金。

  • 可溯追踪

    交易数据由银行全程记录，账目清单随时可查，支持回溯追踪，有效保证投资人合法权益。

内蒙古陕坝银行资金存管系统

用户授权操作，资金完全隔离

1、什么是银行存管？

银行存管是指P2P平台、投资人、借款人在存管银行开设独立存管账户，资金相互隔离，资金的使用、划付在银行存管系统进行，并对资金流向进行管理和监督。

3、实现银行存管后，投资人是否需要到内蒙古陕坝银行开立账户？

每位注册用户在进行投资前，均需要在线开通内蒙古陕坝银行存管账户，无需用户线下前往银行进行开户。

2、实现银行存管有哪些好处？

首先，银行存管实现了平台自有资金和用户资金的分离，避免平台自有资金与用户资金发生混同，当平台自身发生负债、破产时，用户资金不会受到牵连。
其次，银行存管是由银行根据投资人、借款人的指令和合同约定对用户资金的使用、划付进行管理、监督，且由银行对客户资金交易进行记录，避免平台非法动用投资人、借款人的资金，保障投资人、借款人的资金安全。
每位投资用户都要在内蒙古陕坝银行开通独立的个人存管账户，在用户进行投资、提现等涉及资金操作时，均需要跳转至银行存管页面，有效保障用户的资金安全。

4、投资时，用户需要登录内蒙古陕坝银行吗？

在投资前，需要开通内蒙古陕坝银行存管账户，充值时资金由您所绑定的银行卡转入您的存管账户内，不需要登录内蒙古陕坝业银行，用户也无法在内蒙古陕坝银行登录。
