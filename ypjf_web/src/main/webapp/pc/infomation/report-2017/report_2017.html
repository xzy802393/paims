我们17年走过的路

过去的一年，营普金服按照增量平稳、存量控制的规模策略，严控风险，保证每一个项目如期回款，保证每一分投资安全到账。

我们17年走过的路

从2017年第二季度起，营普金服已重点着力于在细分领域挖掘新的资产端，分别在车贷、房贷、企业贷、供应链金融、消费金融方面探索创新，目前已经上线“车贷融”、“房贷融”、“优企融”三款产品。新的金融产品在条件成熟后，将会陆续与大家见面。在产品周期方面，营普金服继续致力于为小微企业及个人解决小额、中短期资金周转流动需求，期限以1个月、3个月、6个月为主。

我们的项目

无论是车贷融、房贷融，还是优企融，营普金服始终坚持“小额分散”的风控原则，为投资人提供收益稳健、风险可控的优质项目，为小微企业及个人提供高效快捷的融资通道。
